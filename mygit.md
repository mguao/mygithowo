## Initialize Git
``` 
git init
```
## get the repo status
```
git status
```
## Add a file/dir to the repo
``` 
git add <file|dir>
```
## Committing from staging area to the repo
``` 
git commit -m "<commit message>"
```
## How to ignore files/dir to the repo
``` 
create .gitignore file in the root of the repo
```
